# Installation

- 1 | Copy the repo

```
git clone https://github.com/Paladynee/layered-generation-rs.git
```

```
cd layered-generation-rs
```

- 2 | Compile & run

Recommended values for depth are 8 to 12

```
cargo run --release YOUR_DEPTH YOUR_SEED
```
